Successful Parts + Need to Improve on:

I think some of the most successful parts of this project came from the
brush settings functions where I was able to create a prefab for sliders and
buttons allowing for me to quickly create interactive features to the editor.
I think for things that need improvement on is the brush styles. I know they
are quite lackluster in originality and will want to improve on it later. I
would also like to fix the problem where using a hotkey command clears the
current selected brush.

What Art Style has Been Created?

The main style is pretty basic with the brushes heavily relying on the default
brush. I really wanted to make a more streamline approach for editing the brush
so that's what I focused on.

Commands:

Backspace : clears canvas to white
O : clears canvas to primary color
L : clears canvas tp complimentary color
, : makes Stroke smaller
. : makes Stroke bigger
1 : default brush
2 : outlined brush
3 : tapering brush
4 : Long multiColor brush
5 : Eraser tool
F : flips color
G : changes between Greyscale and RGB
V : changes Symmetry mode to Vertical
H : changes Symmetry mode to Horizontal
