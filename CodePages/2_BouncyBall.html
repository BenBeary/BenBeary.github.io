Gravity Simulator

Created: 9/13/23

Tags:

ART 101 | JavaScript Coding Run Program Look at Code

I made this as a fun little program to get used to global variables and custom functions. This was made way earlier in the semester before we actually learned this in class. I have experience in other code languages like C#, so this was more for me to figure out the language. Kind of like a warm up

I wanted to try and make it as realistic as possible, so I have variables such as terminal velocity, drag, friction, and of course, gravity.

Simulator uses a "Bouncy Ball"
